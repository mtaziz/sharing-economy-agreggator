get /ads

Request to get list of sharing economy ads

  • Request
  • Response

Query Parameters

  • category: (number)

    ads category by index

    Example:

    1 corresponds to housing
  • zone: (string)

    a city of ads

    Example:

    rouen
  • latitude: (number)

    start point latitude

    Example:

    49.4407
  • longitude: (number)

    start point longitude

    Example:

    1.0964
  • radius: (number)

    radius of lat, lng search by km

    Example:

    10

HTTP status code 200
